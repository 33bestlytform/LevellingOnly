Background Music Directory

This directory contains background music files for the game.
Supported formats: .mp3, .wav, .ogg

To add music to the game:
1. Place your music files in this directory
2. Supported extensions: .mp3, .wav, .ogg
3. The game will randomly select from these files during gameplay

Example:
- background1.mp3
- ambient.wav
- battle_theme.ogg
