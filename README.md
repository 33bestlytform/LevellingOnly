# 我独自升级游戏

一个基于 Pygame 开发的生存射击游戏，玩家需要选择角色，击败一波又一波的敌人，升级自己的能力，坚持10分钟获得胜利。

## 功能特点

- 4个不同特性的角色选择
- 4种武器系统
- 等级提升和技能选择
- 商店系统
- 波次挑战

## 控制方式

- WASD：移动
- Q + 1-4：切换武器
- B：打开商店
- 1-4：在商店中购买物品
- 1-3：选择升级选项

## 安装与运行

### 依赖

- Python 3.7+
- Pygame

### 安装

1. 克隆仓库
2. 安装依赖：`pip install pygame`
3. 运行游戏：`python source.py`

## 游戏机制

- 每波敌人数量随波次增加
- 击败敌人获得金币
- 每波结束后升级并选择技能
- 可以在商店中用金币购买属性提升
- 坚持10分钟获得胜利

## 角色介绍

- 战士：血厚攻高
- 射手：攻速飞快
- 忍者：极致灵活
- 机器人：超级肉盾

## 武器介绍

- 手枪：基础武器，单发伤害
- 霰弹：范围伤害，伤害较低
- 激光：高伤害，速度快
- 剑：近战伤害，范围攻击

## 升级选项

- 生命+3
- 伤害+1
- 攻速提升
- 移速+1

## 商店物品

- 生命+2（10金币）
- 伤害+1（15金币）
- 攻速提升（12金币）
- 移速+1（10金币）
