# LevelingOnly 游戏发行版

## 游戏简介
这是一个独立的生存射击游戏，玩家需要在不断涌来的敌人中生存下去，通过击败敌人获取金币解锁新武器和升级。

## 如何运行游戏

### Windows 用户
1. 下载 `LevelingOnly.exe` 文件
2. 双击运行即可开始游戏
3. 无需安装任何额外软件

### Linux 用户
1. 下载 `LevelingOnly` 可执行文件
2. 在终端中运行以下命令使其可执行：
   ```bash
   chmod +x LevelingOnly
   ```
3. 运行游戏：
   ```bash
   ./LevelingOnly
   ```

## 游戏操作

### 基本操作
- WASD 键：移动
- 鼠标：瞄准
- 鼠标左键：射击（手枪）/拾取武器
- B 键：打开商店
- Q 键：使用终极技能

### 商店系统
- 按 B 键打开商店
- 按 1-4 键购买升级
- 价格会随购买次数递增

### 升级系统
- 清完一波敌人后会出现升级选项
- 按 1-3 键选择升级

## 游戏特色

### 角色系统
- 战士：高生命值，拥有AOE技能
- 射手：高攻速，技能提升武器攻速
- 忍者：高移速，技能提供短暂无敌
- 机器人：技能提供护盾

### 武器系统
- 普通手枪：鼠标点击射击
- 霰弹枪：自动攻击，散射伤害
- 激光枪：自动攻击，穿透伤害
- 圣剑：近战范围攻击

### 敌人类型
- 普通敌人：基础敌人
- 弓箭手：远程攻击
- 史莱姆：会分裂的敌人
- 特殊敌人：带星号标识的强敌（5%概率出现）
- BOSS：每5波出现一次，血量厚，攻击力高

## 系统要求

### Windows
- Windows 7 或更高版本
- 至少 2GB RAM
- 支持 OpenGL 2.0 的显卡

### Linux
- 任意现代 Linux 发行版
- 至少 2GB RAM
- 支持 OpenGL 2.0 的显卡
- X11 显示服务器

## 常见问题

Q: 无法移动怎么办?
A: 关闭输入法或使用英文输入

Q: 游戏运行缓慢怎么办？
A: 关闭其他占用资源的程序，确保电脑配置满足最低要求。

Q: 游戏闪退怎么办？
A: 确保显卡驱动是最新的，尝试以管理员身份运行游戏。

Q: 没有声音怎么办？
A: 检查系统音量设置，确认扬声器工作正常。

此游戏使用以下技术构建：
- Python 3.13
- Pygame 2.5.2
- PyInstaller 打包工具

打包的可执行文件包含了所有必要的依赖库，因此用户无需安装Python或任何其他软件即可运行游戏。
