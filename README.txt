# LevelUpGame 安装说明

## 游戏简介
这是一款Roguelike射击游戏，玩家需要对抗不断涌来的敌人，升级武器和角色属性。

## 系统要求
- 操作系统：Windows 10/11 (64位)
- 内存：至少4GB RAM
- 显卡：支持OpenGL 2.0的显卡
- 存储空间：约100MB可用空间

## 安装步骤
1. 运行 "LevelUpGame_Setup.exe"
2. 按照安装向导提示完成安装
3. 安装完成后可在开始菜单或桌面找到游戏快捷方式

## 游戏操作
- WASD：移动角色
- 鼠标：瞄准方向
- 鼠标左键：射击（单击快速射击，按住限制射速）
- B键：打开商店
- Q键：使用终极技能

## 游戏特色
- 四种独特武器：普通手枪、霰弹枪、激光枪、圣剑
- 多种敌人类型：普通敌人、弓箭手、史莱姆、特殊敌人、BOSS
- 角色升级系统：战士、射手、忍者、机器人
- 特殊敌人：每波有5%几率将普通敌人转化为特殊敌人

## 常见问题
Q: 游戏无法启动？
A: 请确保系统已安装最新的DirectX和Visual C++运行库。

Q: 游戏运行缓慢？
A: 关闭其他占用资源的应用程序，确保有足够的内存空间。

## 卸载方法
通过控制面板或设置应用中的"应用和功能"卸载LevelUpGame。
